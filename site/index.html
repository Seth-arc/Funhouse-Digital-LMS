Grades 4-9 Digital Literacy Platform

Funhouse Digital LMS

A game-based learning management system designed for rural classrooms, with progress tracking, role-based dashboards, and a practical three-station lesson model.

View Lesson Model Quick Start
4 User Roles
3 Learning Stations
15+ Seeded Games

Three-Station Lesson Model

Station 1

Computational Thinking

Problem-solving skills, logic activities, and sequencing practice.

Station 2

Typing

Keyboard fluency with repeatable drills and measurable progress.

Station 3

Purposeful Gaming

Curriculum-aligned games that reinforce literacy and digital confidence.

Quick Start

Run these commands from the project root:

npm run install-all
npm run dev

This starts the backend on http://localhost:5000 and the frontend on http://localhost:3000.
